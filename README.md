## Built With
***
A list of technologies used within the project:
* React
* Typescript
* Express
* MongoDB/Mongoose
* Cloudinary
## Key Features
***
* Login/Signup
* Create articles/posts
* Upload images to posts
* Upload profile image
* Write comments on posts
* User profiles
* Delete posts
* Responsive design
