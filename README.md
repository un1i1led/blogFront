<h1>Blogging App</h1>
A fullstack blogging app project created as a way to learn about REST APIs and mongoDB

<h3>Built with</h3>
- React
- Typescript
- Express
- MongoDB/Mongoose
- Cloudinary

<h3>Key Features</h3>
- Login/Signup
- Create articles/posts
- Upload images to posts
- Upload profile image
- Write comments on posts
- User profiles
- Delete posts
- Responsive
